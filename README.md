# Career Quiz

An interactive career guessing game for teenagers (14–15 years old) to discover different jobs and start thinking about their future.

## How to play

- Read the 3 clues and guess the career from 4 options
- After answering, the career portrait is revealed
- Track your score and streak across all 12 careers
- Use "I give up" if you're stuck — no shame!

## Careers included

Designer · Teacher · Sales assistant · Influencer · Doctor · Architect · Baker · Marketing manager · Firefighter · Engineer · Reporter · Finance advisor

## How to deploy on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your quiz will be live at `https://yourusername.github.io/career-quiz`
